# SC-ELROM - Proiect Baze de Date

Proiect de universitate pentru materia Baze de Date. 

Acest repository conține interfața și structura bazei de date pentru site-ul de prezentare al unei firme fictive de instalații electrice (SC-ELROM).

Platforma este concepută pentru a fi complet statică pe partea de frontend (HTML, CSS, JavaScript) și utilizează Supabase ca Backend-as-a-Service pentru integrarea cu o bază de date PostgreSQL și gestionarea funcționalităților dinamice (cum ar fi autentificarea, stocarea de mesaje de contact, programări de întâlniri, recenzii publice aprobate de un administrator și newsletter).
